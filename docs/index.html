Chrome · Edge · Brave

Manifest V3. Se carga en modo desarrollador.

  • Descarga y descomprime el ZIP.
  • Abre chrome://extensions.
  • Activa el Modo de desarrollador (arriba a la derecha).
  • Pulsa «Cargar descomprimida» y elige la carpeta.
Descargar para Chrome

Firefox

El ZIP se carga como complemento temporal; la instalación permanente requiere un .xpi firmado.

  • Descarga el ZIP.
  • Abre about:debugging#/runtime/this-firefox.
  • Pulsa «Cargar complemento temporal».
  • Selecciona el manifest.json dentro del ZIP descomprimido.
